# 🐚 crab — Agent Shell for Repo Entry/Leave

Crab is the fleet's shell for entering and leaving repos (shells). Like entering a MUD room: step into a repo, use its tools, and step out.

## Tools

- `crab.py` — The crab CLI tool (`python3 crab.py <command>`)
- `install.sh` — Install crab to /usr/local/bin
- `crab-wrapper.sh` — Symlink target that resolves to crab.py

## Provides

- enter-leave-shell
- manifest-parser
- tool-execution
- repo-inspection
